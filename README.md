# MEDATE
## Descripción del proyecto
Proyecto de bicicletas estáticas con animación desarrollado en C# Windows Forms
## Objetivos
- Simular una carrera de bicicletas
- Aplicar conceptos de programación orientada a objetos
- Implementar animaciones en Windows Forms
## Tecnologías utilizadas
- C#
- Windows Forms
- Visual Studio Community
- GitHub
## Instalación
1. Descargar el repositorio
2. Abrir el proyecto en Visual Studio Community
3. Compilar la solución
## Ejecución
1. Ejecutar el proyecto desde Visual Studio
2. Presionar el botón Iniciar Carrera
3. Observar la simulación de la carrera de bicicletas
